# Sendsar SDK for Flutter

Headless chat client for Flutter apps. Use a **session JWT** from your backend — never `sk_*` in the app.

## Packages

| Package | pub.dev | Role |
| --- | --- | --- |
| `sendsar_chat` | [pub.dev/packages/sendsar_chat](https://pub.dev/packages/sendsar_chat) | Connect, rooms, messages, typing, presence, call signaling |
| `sendsar_call` | [pub.dev/packages/sendsar_call](https://pub.dev/packages/sendsar_call) | Voice & video (LiveKit) |

## Getting started

1. Complete server setup — [Quick start](https://docs.sendsar.com/setup/quickstart)
2. Install the SDK — see [sdk/README.md](sdk/README.md)
3. Integrate — [Flutter Setup SDK](https://docs.sendsar.com/sdk/flutter/setup)

## Releases

Tagged releases match pub.dev versions (`v0.6.1`).
