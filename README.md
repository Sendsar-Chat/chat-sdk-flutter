# sendsar_chat

Flutter client for Sendsar realtime chat. Use a **session JWT** from your backend — never `sk_*` in the app.

## Install

```yaml
dependencies:
  sendsar_chat: ^0.1.0
```

```bash
flutter pub add sendsar_chat
```
